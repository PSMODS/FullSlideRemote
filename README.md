# FullSlide Remote Site

Site público do **FullSlide Remote**, preparado para GitHub Pages.

## Publicação no GitHub Pages

1. Suba os arquivos para a branch principal deste repositório.
2. No GitHub, abra `Settings > Pages`.
3. Em `Build and deployment`, selecione:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
4. Salve.

URL esperada:

- `https://psmods.github.io/FullSlideRemote/`

Links públicos principais:

- Home: `https://psmods.github.io/FullSlideRemote/`
- Política de Privacidade: `https://psmods.github.io/FullSlideRemote/privacy-policy.html`
- Suporte: `https://psmods.github.io/FullSlideRemote/support.html`
