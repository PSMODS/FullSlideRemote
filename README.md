# FullSlide Remote Site

Site publico do **FullSlide Remote**, preparado para GitHub Pages.

## Publicacao no GitHub Pages

1. Suba os arquivos para a branch principal deste repositorio.
2. No GitHub, abra `Settings > Pages`.
3. Em `Build and deployment`, selecione:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
4. Salve.

URL esperada:

- `https://psmods.github.io/FullSlideRemote/`

Links publicos principais:

- Home: `https://psmods.github.io/FullSlideRemote/`
- Politica de Privacidade: `https://psmods.github.io/FullSlideRemote/privacy-policy.html`
- Suporte: `https://psmods.github.io/FullSlideRemote/support.html`

- Download estavel do Windows: `https://psmods.github.io/FullSlideRemote/downloads/FullSlideRemote.exe.zip`
