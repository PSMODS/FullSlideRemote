Como funciona

O fluxo foi pensado para qualquer pessoa conseguir usar em poucos passos.

1

Baixe o app do computador

Baixe o pacote do Windows, extraia o arquivo e abra o FullSlide Remote no PC.

2

Conecte o celular

Deixe PC e celular na mesma rede e leia o QR Code ou encontre o PC pela rede.

3

Controle a apresentação

Use o celular para avançar slides, iniciar o PowerPoint e abrir mídia no PC.

Principais recursos

Uma interface simples para controlar o que realmente importa durante a apresentação.

Controle de apresentações

Avance, volte, inicie e encerre apresentações no PowerPoint ou PDF com poucos toques.

Mídia no PC

Abra imagens, vídeos e áudios diretamente no computador, com visualização e biblioteca simples.

Conexão prática

Conecte por QR Code, por descoberta na rede local ou digitando o link do computador.

Novidades da versão 0.2.6

Atualização com interface mais limpa, anotações melhores, QR Code refinado e pareamento mais confiável.

📱 Atualização — FullSlide Remote

✨ Interface mais limpa e profissional

O Host Windows recebeu um visual mais organizado, com dashboard leve, cabeçalho reforçado, cards superiores e ribbon compacto.

Agora:

  • Botões principais com mais destaque e hierarquia visual.
  • Ações Iniciar PPT e Iniciar PDF destacadas como botões principais.
  • Painel Operador do PC reposicionado ao lado do Log remoto.

📝 Anotações melhores

A aba de anotações ficou mais direta para leitura e uso rápido durante apresentações.

  • Aba Anotações sem o cartão grande de status/conexão.
  • Mais espaço livre para leitura.
  • Tela de leitura e lista de anotações respeitam melhor o tema escuro.

📶 Conexão e QR Code refinados

  • Card de conexão do Host mais compacto.
  • QR Code clicável para abrir uma visualização maior.
  • Pareamento mais confiável com reexibição de pedidos pendentes.
  • Tempo para aprovar o celular aumentado para evitar falhas por demora no PC.

🌙 Temas e versões alinhadas

  • Temas claro e escuro refinados para melhor contraste e leitura.
  • Android, AAB e Host Windows alinhados na versão 0.2.6.
